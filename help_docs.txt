***SSD Test Shell Help***
      write         지정한 LBA에 Data를 기록한다.
        Usage) write [LBA] [Data]
      fullwrite     모든 LBA에 Data를 기록한다.
        Usage) fullwrite [Data]
      read          지정한 LBA의 Data를 출력한다.
        Usage) read [LBA]
      fullread      모든 LBA에 대해 Data를 출력한다.
        Usage) fullread
      exit          Test Shell을 종료한다.
      help          도움말을 출력한다.
